Angular 4 cli 筆記

Veröffentlicht am 2017-08-01

angular 指令

ng install -g @angular/cli
**//安裝 angular cli 套件**

* ng new hello
// 建立一個 hello 的目錄，透過 npm 安裝相關的套件

* ng serve
// 啟動 http 伺服器 localhost:4200

* ng g c component-name
// g => generate   c => component
// 建立一個 component-name 的 folder 然後新增相關的檔案
// 註冊到 app.module

* ng g p pipe-name
// p for pipe
// 建立 Pipe 相關的檔案
// 註冊到 app.module

* ng g s service-name
// g => generate s => service

安裝 bootstrap

npm install bootstrap --save

在 style.css 在開頭加上

component

Component 是最小的區塊
Module 是 componet 的 container

一個 App 最少有一個 Component 跟 Module

package.json 的版本數字 ^3.3.7
major.minor.patch
^ 代表可以使用 major 相同的 major 版本的 3.3.8 或 3.3.9 但不能用 4.1 major 版本不同的

Directive (指示)

*ngFor="let c of course"

when using directive to modify structure of DOM adding or remove element , add prefix *

Dependency Injection

app.module 在 @NgModule 的 providers 中註冊要被 DI 的元件

> @Injectable() // 代表可被注入的 Service

Binding

DOM style object

雙向繫結

**Two binding [()] banana in a box 用來記憶語法**

ngModel 是 angular 的雙向繫結語法

**需要在 app.module.ts 引用 FormsModule.**
**import {FormsModule} from '@angular/forms';**
**在 @NgModel 的 imports 屬性陣列中加入 FormsModule**

Build-in pipe

DatePipe at Angular.io

Custom pipe

pipe 跟 component 在 module 中是相同的地位，需要跟 module 註冊

Component API

input property
import { Input } from ‘@angular/core’;

@Input('is-favriote') > decorate a input properties and call it like a function
// is-favriote 是別名(alise - nickName)設定

別名可以讓 template 與 ts 之間的耦合降低，不因為ts 批次改名而出錯. 
keep the contract of your components stable

output event
import { Output,EventEmitter } from ‘@angular/core’;

@Output() change=new EventEmitter( {newValue : this.isSelected} );
在 html 的 event 新增一個 $event object 參數，代表 @Output 傳入的 object 參數

可以將 object 參數的 object 型態寫成 interface 
export 讓外部的 component 引用，形成共用

Shadow DOM

a specification (規範) enables DOM tree and style encapsulation(膠囊化).

Allows us to apply scoped styles to elements
without bleeding out to outer world.
